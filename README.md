# Tombola Masterclass

Static product site and draft product content for Tombola Masterclass.

## Deploy on Netlify

- Import this repo into Netlify
- Build command: leave empty
- Publish directory: `.`
- Netlify Forms is already configured in `index.html`

## Product files

- `index.html` — sales page
- `styles.css` — page styling
- `script.js` — order capture submission handler
- `tombola-masterclass-guide.md` — main guide draft
- `tombola-masterclass-assets.md` — templates and checklists draft
